Task Manager

in this task manneger you will have 4 Options:
new task:
by presing the "Add task" button you will add a new task to your task maneger.
Delete:
by pressing the "delete" button you will delete the task that you chose.
Edit:
by pressing the "edit" button a prompt will jump for you to change the description of this task.
complete:
by preesing the "complete" button your task will be marked in a green color,for you to know you completed it.
